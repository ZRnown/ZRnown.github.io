PicGO的使用，让你写Markdown插入图片更方便！

  当你写 markdown 时有没有发现 Chrome 开个图床，既浪费内存又麻烦不便捷？如果是这样，那说明你肯定需要这个！

废话

  对，这就是超 6 的用于快速上传图片并获取图片 URL 链接的工具(官方是这么解释的)它叫PicGO

  有了它，你可以通过快捷键快速把剪切板里的图片上传到图床上，然后 markdown 格式的图片会呗拷贝到你的剪切板里，没错，你只需要敲快捷键然后你就可以快速为你的 markdown 文档添加图片 🌶！4 不 4 很厉害。

  这是他的官方文档，看了这个你就会使用 🌶！但是如果你不想点的话，接下来我会教你创建 github 图床和如何使用它！

正题

我目前使用的是 github 图床，挺方便挺好用，速度也跟得上(就个人而言)
不废话了，明明进入了正题却还没开始 😒

下载

你可以在PicGO这里下载，目前我这里的最新版本是 2.2.2(很 2 对吧)，下载方法很简单，点点点就是了。

github 账号注册

下载完事儿你不用管，咱先得有个 github 账号，如果你已经有了你可以跳到下一步。
注册方法很简单。

打开 www.gitub.com。

对，就这样，右上角有一个 Sign up(注册)，点进去

这里分别输入你的用户名邮箱密码，确认之后会发给你个验证码，你验证后就完事儿了。

创建仓库

首先这里你要 new 个仓库，看下图左上角有个 new，点进去。

然后再照着我图上的输就行了。这是在创建个仓库用于存放图片。然后你要记住你的仓库名，仓库名=Owner/Repository name。我的用户名是 ZRnown，Respository name 是 Picgo，那仓库名就是 ZRnown/Picgo 了，这个要注意，等下会用到。

创建 token

你现在有个空仓库了，你现在要复制你现在的浏览器地址栏上的地址（等下要用上），然后点你的头像，再点 settings

再分别点 Developer settings，和 Personal access tokens。

点击 Generate new token 创建 token

然后名字填个 PicGo，咱暴力点，底下全点满 🤣，再 Generate token 就完事儿了咱。

完事之后别急着离开，复制一下这一堆看不懂的东西。

PicGO 配置

点开咱刚下载的 PicGO，你会看到是这样的。


咱先点击 PicGo 设置。

点击修改快捷键，这里我使用的是 Ctrl+Shift+c，你可以挑个你喜欢的快捷键。

接着咱来设置图床，点击图床设置，再点击 github 图床。

这里仓库名填创建仓库，那一步的仓库名，分支名默认填 master，token 填刚才新建 token 那里的一些看不懂的东西。指定路径随便填，你也可以像我这样填 img/，这样你上传图片的时候，它会在你的 github 仓库下创建一个 img 文件夹，然后在里面存放你的图片。设定自定义域名就是 https://raw.githubusercontent.com/你的用户名/仓库名/master。 我的是 https://raw.githubusercontent.com/ZRnown/Picture/master，然后点设为默认图床并确认。完事儿了咱！

以后在你复制一个图片并敲上你的快捷键之后，就会有如下提示。然后你再直接 Ctrl+v，它自己就会把 markdown 格式的图片粘贴出来，是不是超爽超方便！！！
